# beluga-core

#### 介绍
爱尚生活家旗下核心类库

#### 软件架构
该核心类库主要解决重复造轮子的问题，通过引用该核心类库，实现平时使用的算法，功能，服务。
#### 安装教程
在pom.xml文件中加入maven库
```
 <repositories>
        <repository>
            <id>beluga-core</id>
            <url>https://raw.github.com/guocjsh/beluga-core/dependency</url>
        </repository>
 </repositories>
```
#### 使用说明
加入maven引用即可使用类库
```
  <dependency>
        <groupId>com.beluga.core</groupId>
        <artifactId>beluga-notice</artifactId>
        <version>1.0</version>
  </dependency>
```
