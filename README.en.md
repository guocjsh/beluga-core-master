# beluga-core

#### Description
Love is still home of the core class library

#### Software Architecture
The core class library mainly solves the problem of repeating the wheel, by referring to the core class library, to achieve the usual use of algorithms, functions, services.

#### Installation
Add the Maven libraries to the pom.xml file
```
 <repositories>
        <repository>
            <id>beluga-core</id>
            <url>https://raw.github.com/guocjsh/beluga-core/dependency</url>
        </repository>
 </repositories>
```
#### Instructions
Add maven references to use the class library
```
  <dependency>
        <groupId>com.beluga.core</groupId>
        <artifactId>beluga-notice</artifactId>
        <version>1.0</version>
  </dependency>
```
